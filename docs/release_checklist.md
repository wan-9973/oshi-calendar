## ライブ環境でのみ確認可能な残項目（認証情報設定後に実施）

1. **成人向け除外ジャンルIDの確定**: BooksGenre/Search・市場ジャンルAPIで該当IDを取得し `config.EXCLUDED_BOOKS_GENRE_PREFIXES` / `EXCLUDED_ICHIBA_GENRE_IDS` に設定。
2. **リクエスト間隔の本番実測**: アクセスログで1秒以上を確認（uvicornワーカー1プロセス運用が前提）。
3. **実在推し3件でのライブ検索テスト**（Phase 0残・関連度閾値0.4の妥当性確認）: `python -m src.cli search "推し名"`
4. **7日間の無人試験運用**（Phase 3完了条件）: cron設定後、crawler/retentionのログで確認。
5. **Ichiba 2026-04-01版の実レスポンス確認**: フィールド構成が変わっている場合は`search_service._normalize`のフォールバックチェーンが吸収し、差分がログに出る（§12方針）。
